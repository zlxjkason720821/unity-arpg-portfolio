# 成品验收清单

## Play 战斗（主演示）

1. 打开 `Assets/Scenes/Demo` → Play  
2. 应看到：蓝玩家、一圈红敌人、顶部操作提示、底部技能栏、Bloom 光感  
3. `WASD` 移动，镜头跟随  
4. `J`/左键普攻，`Q` Cleave，`E` Shockwave（有特效球 + CD）  
5. 敌人受击闪色/描边变粗，死亡溶解消失  

## 编辑器（简历加分段）

1. `ARPG Tools > Create Sample Data`  
2. `ARPG Tools > Skill Editor`：拖 Damage，柱高与数字变化  
3. `ARPG Tools > Level Editor`：Place Enemy → Scene 点击 → Save  

## Console

无红字即可（Input Manager 黄字可忽略）。
